# @textforge/tables

`@textforge/tables` owns TextForge's richer table UX:

- CSV/TSV structured grid editing with explicit apply/discard boundaries
- ITM catalogue and relationship-matrix table surfaces built on TanStack Table
- grouped diagnostics / issue tables for resource and document-context review

The package exports reusable helpers for delimited parsing/serialization, table model
construction, and the contribution manifest consumed by the TextForge workbench.
