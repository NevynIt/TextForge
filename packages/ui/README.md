# @textforge/ui

Shared React workbench primitives for TextForge.

Phase 3.4 adds the local resource-badge renderer, compact inspector and empty-state cards, calmer command chrome, and the drawer-style utility shell behavior. `@textforge/ui` owns the bundled `lucide-react` dependency for generic React icon affordances such as folder/file/search/warning controls; resource identity itself stays in the deterministic badge token rather than a per-file icon taxonomy.
