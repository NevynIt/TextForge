# TextForge User Guide

## Lua Console Features

- Run quick snippets, the active Lua document, or selected Lua text.
- List registered Lua actions.
- Inspect the active pipeline catalog for the current document.
- Inspect the read-only registry snapshot available to an elevated power session.
- Open the previous result as a generated workspace file.
